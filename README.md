Welcome to BookStore System Project

In order to run server for this particular project, You need to create .env file for each microservice first in order to add two private things:
- JWT_SECRET
- MONGO_URI

If you want to deploy it on EC2 Instance, you can create image and push it to docker hub first. After that you can create EC2 instance and put public ip and then you can pull image from docker hub and run.
